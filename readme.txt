En este trabajo se busco mas enfasis en el uso de tablas, etiquetas como tr,td,th, etc.
Se usan hovers con fin de "animaciones" y en la tabla tambien cambiando su respectivo color de fondo.
Se uso flexbox principalmente en la estructura de las paginas.
En las tarjetas creadas se uso el wrap ya teniendo como base el tp02, donde utilizamos la misma estructura pero solo con imagenes.
El contenido si bien no es de primera, se valora el codigo semanticamente correcto y estructura conseguidas, por eso uso lorem para ahorrar tiempo.
Pudiendo usar selectores se facilito el trabajo sin mezclar codigo y pudiendo reutilizarlo mucho, ya que al tener un codigo que no se repite solo tuve que cambiar el main (tanto estructura como contenido) en cada pagina.
Se uso ya enlaces a la propia pagina por medio de los ID.
